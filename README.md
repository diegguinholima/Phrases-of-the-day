# Phrases-of-the-day
First application with React Native

(/img/telaStart.jpg)
![Phrases-of-the-day](/img/telaFrase.jpg)

## Setup

```
git clone "https://github.com/diegguinholima/Phrases-of-the-day"
```

```
npm i
```

### Run Android

```
cd Phrases-of-the-day
```
```
react-native run-android
```

### Run IOS

```
cd Phrases-of-the-day
```
```
react-native run-ios
```
